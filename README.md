Настоящий сайт содержит описание различных техник и методик эффективного обучения, изображения, видео и анимацию.

Для определения структуры и описания содержания веб-страницы использовался HTML. Файловая структура проекта организована по классической БЭМ-технологии Nasted. Общая структура страницы размечена семантическими тегами.

Внешний вид веб-контента был описан с помощью средств CSS. Использованы технологии и инструменты, такие как Flexbox, Grid Layout, Iframe, и другие.
